Algorithms

To provide an example of how datasets can be used, we implemented a point-in-polygon algorithm that filters location data by a polygon. Likewise, we designed a simple Machine Learning model for augmenting the datasets with weather information.

Filtering data by polygon

By providing a CSV file with location coordinates and a text file with the coordinates of the desired polygon region, the point in polygon algorithm creates a CSV file containing only the data from locations inside the provided polygon region.

For each point in the dataset, the algorithm checks in linear time the number of intersections of an "infinitely" long horizontal line starting at that particular point with the edges of the polygon. If the number of intersections is odd or the point lies on an edge of the polygon, the point is added to the filtered dataset. Otherwise, the point is outside the region and thus ignored.

ML weather prediction model

The ML model was implemented as a regression algorithm, with the average wind speed as the explained (dependent) variable, and the month and year as the regressors (independent variables).

Two models were provided: a linear regression, and an SVM model. For both models, the data was normalised by subtracting the mean and dividing by the standard deviation. We used 70 % of the data for the training process, and 30 % for testing the performance.

Because the relationship between the variables is unlikely to be linear, we used the SVM model, with a radial basis function kernel, as the main algorithm. The "kernel trick" allows performing non-linear classification by mapping the inputs into high-dimensional feature spaces. The linear regression model was used as a means of comparing the performances.

Note: Because AI analysis was not the scope of our project, and due to the lack of relevant data, the model was designed as a proof of concept. We acknowledge that our model is neither complete nor significant. As a consequence, the results were not meaningful: the mean absolute error was 1.4996 and the R^2 score was 0.3384.
