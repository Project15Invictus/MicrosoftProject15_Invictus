FAD Open Datasets
UI Design

High usability and efficiently system and UI design matters a lot. A good structure of the system can be beneficial to our user, and for our goal to help data scientists, a friendly UI is necessary.

Design Principles

Simplicity and Usability.

Our target users are data scientists without high computer science skills. Follow the principle of user-centerd design, we need an UI that is friendly and easy to use. An UI with lots of scripts (e.g. bash shell) is not ideal. We provide a graphical UI which is much simpler to use than the scripts UI for data scientists. Users may achieve what they need by a few mouse clicks, and the functionalities of each part of the UI is easily understandable.
